# Exercises - Day 8

## Exercise 1 - Clean the User's Name

### Goal
Ask the user for their name, remove any extra whitespace, capitalize each word, and print a welcome message.

### Requirements
- Use `input()`
- Use `.strip()`
- Use `.title()`
- Use `print()`

**Example Output**
```
What's your name?     aBdO saLAh
Welcome, Abdo Salah!
```

## Exercise 2 - Add Two Numbers

### Goal
Ask the user for two decimal numbers and print their sum.

### Requirements
- Use `float()`
- Store the numbers in variables
- Print the result

**Example Output**
```
What's the 1st number? 4.5
What's the 2nd number? 3

The result is: 7.5
```

## Exercise 5 - Mini Challenge

### Goal
Create a small program that:

1. Asks the user for their name.
2. Cleans the input using string methods.
3. Asks the user for their age.
4. Converts the age to an integer.
5. Prints:

```
Hello, Abdo!
You are 20 years old.
```

### Bonus ⭐
Print the output like this:

```
Hello -> Abdo
Age -> 20
```
