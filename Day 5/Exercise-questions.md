# Exercise 1: Random Color Picker

## Objective
Practice using `import random` and `random.choice()`.

## Instructions
1. Import the `random` library.
2. Create a list containing these colors:
   - Red
   - Blue
   - Green
3. Randomly choose one color from the list.
4. Print the selected color.

### Example Output
```
Blue
```


# Exercise 2: Random Pet

## Objective
Practice using `random.choice()` with a different list.

## Instructions
1. Import the `random` library.
2. Create a variable called `pet`.
3. Use `random.choice()` to randomly choose one pet from this list:
   - Cat
   - Dog
   - Rabbit
   - Fish
4. Print the selected pet.

## Example Output

```text
Dog
```

or

```text
Rabbit
```

## Bonus Challenge ⭐

Instead of printing only the pet, print a sentence like:

```text
Your pet is: Cat
```
