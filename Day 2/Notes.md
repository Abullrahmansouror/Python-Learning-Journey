**if**
- It's a conditional statement in Python. It lets your program make a decision.
Example :
if x > y: 
    print ("x bigger than y")


**elif**
- If the previous condition was False, check another condition.
Example :
if x > y: 
    print ("x bigger than y")
elif x < y:
    print("x less than y")


**else**
- If none of the conditions are True, run this code.
Example :
if x > y: 
    print ("x bigger than y")
elif x < y:
    print("x less than y")
else:
    print("x equal to y")


**or**
- Checks multiple conditions. If at least one condition is True, the result is True.
Example :
if x > y or x < y:
    print("x not equal to y")
else: 
    print("x equal to y")


**and**
- and checks multiple conditions. All conditions must be True for the whole condition to be True.
Example :
if x > y and x < y:
    print("x not equal to y")
else: 
    print("x equal to y")

- It's a type of data in python like string, inteeger and float. This Bool has just true and fulse output.
