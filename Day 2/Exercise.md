**Exercise 1 - Comparison Operators**

Ask the user for two numbers.

Print:

"x is bigger than y" if x > y
"x is less than y" if x < y
"x is equal to y" if x == y


**Exercise 2 - or**

Ask the user for a number.

If the number is 5 or 10 or 15, print:

Lucky number!

Otherwise print:

Try again!


**Exercise 3 - and**

Ask the user for their age.

If the age is between 18 and 30, print:

You can join.

Otherwise print:

You cannot join.


**Exercise 4 - if, elif, else**

Ask the user for a score.

If the score is 90 or more, print "Grade A"
Else if the score is 80 or more, print "Grade B"
Else if the score is 70 or more, print "Grade C"
Else print "Failed"


**Exercise 5 - Mixed Challenge ⭐**

Ask the user for:

Username
Password

Rules:

Username must be "admin" and password must be "1234" → Print:
Login successful.
Else if the username is "admin" or the password is "1234" → Print:
Almost correct.
Else print:
Access denied.
