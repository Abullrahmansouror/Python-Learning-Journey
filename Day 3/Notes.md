## 1- What I have learned :

**loops**
- loop: Repeats code a fixed number of times or for each item in a sequence (like a list or range()).


**while**
- While loop: Repeats code as long as a condition is True. It stops when the condition becomes False. 
Example:
    x = 1
    while x <= 5:
    print("Hello")
    x = x + 1

**for**
- A for loop repeats code for each value in a sequence, such as numbers from range().
Example: 
    for i in range(1, 6):
    print(i)
